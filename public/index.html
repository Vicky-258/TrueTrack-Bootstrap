TrueTrack

A local-first, self-hosted media intelligence tool that runs entirely on your machine and opens in your browser. No cloud. No accounts. No background services.

What happens when you install this?

  • Downloads the open-source TrueTrack code to your machine
  • Installs required dependencies with your permission
  • Runs a local server on your system
  • Opens the app in your default browser
  • Nothing runs in the background when you stop it

After installation

  • Start TrueTrack using the desktop launcher or the truetrack command
  • The app opens in your browser at http://localhost:8000
  • Stop it anytime with Ctrl + C

Security & Transparency

  • No telemetry
  • No auto-start on boot
  • No background services
  • Fully open source
  • Easy uninstall: delete the install folder

This installer only clones the repository and hands control to the real installer. You can audit it in under a minute.

Who is this for?

  • People who want local-only tools
  • Self-hosters and power users
  • Anyone tired of cloud-locked software
